What is URL Encoding?

URL Encoding, or Percent-encoding, is the standard for encoding information in a Uniform Resource Identifier (URI). Characters not allowed in a URL are replaced by a % followed by their hexadecimal equivalent.

Why use this?

URLs only support a subset of ASCII. Spaces, emojis, or symbols like & can break links if not properly encoded for the browser.

Common Codes

The most common code is %20 (Space). Others include %21 (!), %23 (#), and %26 (&).

Encoding vs Encryption

Encoding reformats data for system compatibility; it is not a security measure and can be easily reversed by any decoder.
