What are HTML Entities?

HTML Entities are strings of text used to represent reserved characters or invisible symbols in HTML. For example, the browser interprets the < symbol as the start of a tag. To display it as text, you must use its entity equivalent: &lt;.

Why escape HTML?

Escaping code prevents "Cross-Site Scripting" (XSS) attacks and ensures that your browser renders code snippets as text rather than trying to run them as active elements.

Common Entities

The most common are: &lt; (<), &gt; (>), &amp; (&), and &quot; (").

Is this tool offline?

Yes. Like all ToolForge utilities, this processing happens entirely in your browser. No data is sent to a server.

What is "Unescaping"?

Unescaping is the process of turning entities back into their original symbols. This is useful when you are reading raw source code from a database.
