Control Your Crawl Budget

Search engines like Google assign a "Crawl Budget" to every website. If your bot spend time indexing useless pages like your login screen or temporary search results, it might miss your high-value content. A Robots.txt file ensures bots focus only on what matters.

Privacy vs. SEO

It is a common mistake to think Robots.txt hides pages from the public. It doesn't. It only tells "well-behaved" bots not to list them in search results. For true privacy, you should always use password protection or No-Index tags alongside this file.

Where do I put this file?

The robots.txt file must be placed in the **root directory** of your website (e.g., https://yourdomain.com/robots.txt). Bots will not look for it in subfolders.

What does "User-agent: *" mean?

The asterisk (*) is a wildcard that applies the following rules to **all** web crawlers (Google, Bing, DuckDuckGo, etc.). If you want to target a specific bot, you would use User-agent: Googlebot.
