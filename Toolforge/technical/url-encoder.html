What is URL Encoding?

URL Encoding, also known as Percent-encoding, is a mechanism for encoding information in a Uniform Resource Identifier (URI). Characters that are not allowed in a URL must be replaced by a % followed by their two-digit hexadecimal equivalent.

Why do I need this?

URLs can only contain a limited set of ASCII characters. If your link contains spaces, brackets, or non-English symbols, the browser might break. Encoding ensures the link is "safe" for the internet.

What is %20?

The code %20 is the encoded version of a standard space. You will see this often in file paths or search queries.

Is it secure?

Yes. ToolForge processes your URLs entirely in your browser. We never track the links you encode or send them to a server.

Encoding vs Encryption?

Encoding is NOT encryption. It is a way to reformat data so it can be read correctly by systems. It can be easily reversed by anyone using a decoder.
