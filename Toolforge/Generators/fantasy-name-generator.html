Mastering Fantasy Character Naming

01. Phonetic Consistency Elven names use soft vowels (A, E, I) and liquids (L, S), while Dwarven names use hard stops (K, G, D).
02. Descriptive Surnames Surnames like "Stonefist" or "Brightleaf" instantly communicate a character's history or deeds.
03. Say it Aloud If a name is hard to pronounce during an intense D&D session, it might be too complex for a table favorite.
